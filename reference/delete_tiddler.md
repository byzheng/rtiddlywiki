# Delete a tiddler

Delete a tiddler

## Usage

``` r
delete_tiddler(title, bag = TW_OPTIONS("bag"))
```

## Arguments

- title:

  title of the tiddler to retrieve

- bag:

  string defining which recipe to write to (optional, defaults to
  "default")

## Value

no return values

## Examples

``` r
if (FALSE) { # \dontrun{
delete_tiddler("GettingStarted")
} # }
```
