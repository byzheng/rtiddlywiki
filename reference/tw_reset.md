# Reset global options for pkg

Reset global options for pkg

## Usage

``` r
tw_reset()
```

## Value

the default options

## Examples

``` r
tw_options()
#> $host
#> [1] "http://127.0.0.1:8080/"
#> 
#> $http_x_auth_key
#> [1] ""
#> 
#> $recipe
#> [1] "default"
#> 
#> $bag
#> [1] "default"
#> 
```
